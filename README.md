# Average Calculator

Простой проект на Python для вычисления среднего арифметического значения списка чисел.

## Описание

Этот проект был создан для демонстрации практик рефакторинга и работы с системой контроля версий Git и платформой GitHub.

## Исходный код

Исходная, "унаследованная" версия кода находилась в файле `average.py`.

## Процедура рефакторинга

Были проведены следующие изменения для улучшения читаемости, поддерживаемости и надежности кода:

1.  **Добавление документации:**
    - В начало модуля добавлена строка документации с общим описанием его назначения.
    - Функция `calculate_average` была снабжена подробной docstring в формате, который понятен как людям, так и IDE. В ней описаны аргументы, возвращаемое значение и поведение функции.

2.  **Введение аннотаций типов (Type Hints):**
    - Для аргумента функции `numbers` указан тип `list[float]`.
    - Для возвращаемого значения указан тип `float | None`.
    - **Польза:** Повышает ясность контракта функции, помогает в разработке (автодополнение в IDE) и позволяет использовать инструменты статического анализа для выявления ошибок.

3.  **Улучшение проверки входных данных:**
    - Конструкция `if len(numbers) == 0` была заменена на более идиоматичную для Python `if not numbers`.
    - **Польза:** Код становится короче, чище и легче читается.

4.  **Применение принципа раннего возврата (Guard Clause):**
    - Проверка на пустой список и возврат `None` вынесены в самое начало функции.
    - **Польза:** Уменьшает вложенность кода (избегаем "адских" уровней отступов). Основная логика функции теперь находится на одном уровне, что упрощает ее восприятие.

5.  **Изменение точки входа скрипта:**
    - Код для примера использования и тестирования был обернут в блок `if __name__ == "__main__":`.
    - **Польза:** Этот код будет выполняться только тогда, когда файл запускается напрямую, а не когда он импортируется как модуль в другой скрипт. Это важная практика для создания переиспользуемых модулей.

6.  **Добавление базового тестирования:**
    - В блок `__main__` добавлены простые проверки, которые автоматически проверяют корректность работы функции на известных данных.
    - **Польза:** Позволяет быстро убедиться, что рефакторинг не сломал функциональность. В будущем это можно заменить на полноценные unit-тесты (например, с использованием `pytest`).

## Как использовать

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/your-username/python-average-calculator.git
    ```
2.  Перейдите в папку проекта:
    ```bash
    cd python-average-calculator
    ```
3.  Запустите скрипт:
    ```bash
    python average.py
    ```
